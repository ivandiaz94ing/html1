Etiquetas Semanticas Estructurales

div: Es una etiqueta contenedora que no tiene semantica
header: Es la cabezera de un sitio web o de una seccion
main: Define la seccion principal del documento, sólo puede existir una etiqueta main por documento
footer: Pie de pagina de un sitio web o de una de seccion
nav: Representa una navegacion
article: Representa una seccion autocontenido (que por si sola se explica)
aside: Representa contenido complementario o secundario
section: Representa una seccion de contenido generica
address: Representa información de contacto

Etiquetas de Bloque vs Etiquetas de Linea

la etiqueta de bloque por excelencia es el div, una etiqueta de bloque es aquella que genera saltos de linea con sus etiquetas hermanas.
La etiqueta de linea por excelencia es el span, Una etiqueta de linea sólo ocupa el espacio necesario que tiene su contenido. Este es otro span
